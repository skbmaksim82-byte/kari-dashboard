📖 Как работать с дашбордом
☁️ Обновление данных
☁️ Нажмите кнопку ☁️ Обновить в шапке — дашборд загрузит актуальные данные из паспорт магазина.xlsx с GitHub, планировки из папки plans/ и фото рекламы из папки Foto Reklama/. Статус каждого источника отображается в окне в правом нижнем углу.
📋 Данные при старте загружены из последней сохранённой версии xlsx. Чтобы увидеть свежие изменения — нажмите Обновить.
🔍 Поиск и фильтры
🔍 Поле поиска в левой панели ищет по названию, городу, номеру, подразделению, юрлицу. При вводе появляется выпадающий список — выбор из него открывает паспорт магазина напрямую.
🎛️ Фильтры Подразделение / Категория / Вид / Юрлицо / Магазин работают совместно и сужают список в левой панели.
🏷️ Кнопки Зоны оформления фильтруют магазины по наличию конкретной зоны: Торцы, Витрина, Ювелирный отдел и т.д.
📊 Разделы дашборда
📊 Обзор — сводные карточки по зонам оформления. Клик на карточку открывает детальный список магазинов по этой зоне с возможностью сортировки и экспорта в Excel.
📋 Сводные данные — полная таблица всех магазинов со всеми атрибутами. Клик по заголовку — сортировка. Клик по строке магазина — открытие его паспорта. Кнопка 📊 Выгрузить в Excel — экспорт.
🛒 Сводный заказ — формирование заказа ПОСм по магазинам. Переключайтесь между листами (Листовки, Чек-поинты и др.) и вводите количество боксов. Итоги — в строке снизу. Экспорт — в CSV.
🏪 Паспорт магазина — детальная карточка выбранного магазина: все зоны оформления, площади, кассы. Кнопка 🗺️ Планировка открывает PDF-план с зумом и перетаскиванием.
💡 Полезные детали
№ В списке магазинов слева и в выпадающем списке магазины отсортированы по номеру (от меньшего к большему).
⭐ Магазины со статусом TOP 20 отмечены значком в списке и карточке.
🗺️ В планировке: колёсико мыши — зум, перетаскивание — смещение. Кнопки − + и ⛶ — масштаб и вписать в экран. Кнопка ⬇ Скачать PDF — сохранить файл.
📁 Для обновления данных загрузите новый паспорт магазина.xlsx в корень репозитория GitHub, планировки — в папку plans/ (имя файла = номер магазина), фото — в Foto Reklama/ (имя файла = SKU). Затем нажмите Обновить.
